# Image Classification using CNN
## Project Description
This project compares the performance of a Traditional CNN and a Customized CNN for image classification.
## Technologies Used
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn
## Models
1. Traditional CNN
2. Customized CNN
## Dataset
Image dataset containing multiple image classes.
## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
## How to Run
1. Install requirements
pip install -r requirements.txt
2. Run
traditional_cnn.ipynb
or
custom_cnn.ipynb
## Results
Customized CNN achieved better performance than the Traditional CNN.
